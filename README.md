##  Project Overview

This project presents an end-to-end Cyber Security Threat Intelligence analytics solution using Python and Power BI.

The objective is to analyze cyber security incidents, identify attack trends, evaluate financial losses, discover high-risk sectors, and provide actionable business intelligence for decision-makers.

The project covers the complete analytics lifecycle including data cleaning, exploratory data analysis, feature engineering, KPI development, risk assessment, and dashboard creation.


#  Objectives

- Analyze cyber security incidents
- Identify high-risk attack types
- Evaluate financial losses
- Discover vulnerable cities
- Identify high-risk business sectors
- Perform Risk Segmentation
- Build Interactive Power BI Dashboard
- Generate Business Intelligence Insights

#  Key Insights

- Financial sector experiences the highest losses.
- Metro cities are more vulnerable.
- Data Breach and Ransomware create maximum financial impact.
- High-risk incidents contribute significantly to total losses.
- Risk segmentation improves decision making.
